Murm UI

A zero-framework TypeScript chat interface for LLM apps. It handles the browser UI, streaming rendering, chat history, and common controls while your app provides the model provider and storage.

Open demo Read docs View source
What does Murm UI give me?
A chat shell with streaming messages, markdown rendering, attachments, editing, copy actions, and persistent sessions.
Do I bring the backend?
Yes. Plug in your provider and storage, then keep the UI boring.

Small README, fuller docs

The README stays short. The docs page covers install, HTML structure, CSS imports, providers, plugins, storage, and browser support in one place.

Static demo

The demo is built into the Pages artifact with a mock provider, so it works without a backend or API key.
